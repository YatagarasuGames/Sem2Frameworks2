# Модульная система

Это демонстрационное приложение на .NET, использующее модульную архитектуру для управления игровыми очками и таблицей лидеров.

## Инструкция для запуска

1. **Склонируйте проект**
   ```bash
   git clone https://github.com/YatagarasuGames/Sem2Frameworks2
   ```

2. **Перейдите в папку проекта**
   ```bash
   cd Sem2Frameworks2
   ```

3. **Запустите приложение**
   ```bash
   dotnet run
   ```
   *Приложение запустится в консольном режиме и последовательно выполнит действия всех подключенных модулей.*

---

## Особенности архитектуры

Проект разделен на **Ядро** и **Модули**:

* **Ядро (Core):** отвечает за поиск модулей, построение графа зависимостей (алгоритм Кана) и внедрение зависимостей (DI). Оно ничего не знает о логике игры.
* **Модули:**
    * `CoreModule` — регистрация хранилища.
    * `GameModule` — симуляция игрового процесса.
    * `PlayerScoreModule` — расчет бонусов.
    * `LeaderboardModule` — формирование и вывод итоговой таблицы.

---

## Проверка конфигурации

Состав и порядок запуска модулей можно менять в файле `appsettings.json`:
```json
{
  "Modules": [
    "Core",
    "Game",
    "PlayerScore",
    "Leaderboard"
  ]
}
```
*Если убрать обязательный модуль или создать циклическую зависимость, система выдаст понятную ошибку при старте.*
